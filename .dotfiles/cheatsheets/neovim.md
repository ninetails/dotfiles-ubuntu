# Neovim

## Custom

```
<Tab> - change between buffers

<leader>ss - save current file
<leader>zz - save & close current file
<leader>q  - close current file without exit
<leader>qa - exit current file without saving

<leader>ev - open .config/nvim/init.vim
<leader>sv - source neovim config
<leader>ez - open .zshrc
<leader>ea - open .aliases
<leader>cv - open this cheatsheet file
<leader>pi - run PlugInstall

<leader>y - yank to Xorg clipboard 
<leader>p - paste from Xorg clipboard 
```
